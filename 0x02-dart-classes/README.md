# Dart - Classes

## Introduction
This project introduces Dart classes - the foundation of object-oriented programming. You'll learn to create classes, use constructors, convert objects to JSON, implement private properties, and work with inheritance.

## Learning Objectives
- Create basic classes with properties and methods.
- Use constructors to initialize class objects.
- Convert objects to/from JSON using `toJson()` and `fromJson()`.
- Implement private properties using underscore prefix.
- Apply inheritance by extending one class from another.

## Requirements
- All files will be interpreted/compiled on Ubuntu 20.04 LTS using Dart 3.x.x
- All files should end with a new line
